Pin mapping:

Component-----------Pin

- Mode button-------D14---(switch between LED modes)
- Reset button------D33---(Resets the system to OFF mode )
- LED 1 (yellow)----D26
- LED 2 (blue)------D4
- LED 3 (red)-------D5
- OLED SDA----------D21---(Serial Data Line)
- OLED SCL----------D22---(Serial Clock Line)
