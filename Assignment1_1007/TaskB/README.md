Pin mapping:

Component-----------Pin

- Button------------D25---(detects the short and long press)
- LED 3 (red)-------D5----(toggles on short press)
- Buzzer------------D33---(beeps on long press)
- OLED SDA----------D21---(Serial Data Line)
- OLED SCL----------D22---(Serial Clock Line)
